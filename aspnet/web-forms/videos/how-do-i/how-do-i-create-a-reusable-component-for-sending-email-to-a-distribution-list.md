<a name="how-do-i-create-a-reusable-component-for-sending-email-to-a-distribution-list"></a>[Gewusst wie:] Erstellen Sie eine Wiederverwendbare Komponente zum Senden von E-Mail an eine Verteilerliste
====================
durch [Chris PEL-Spareinlagen](https://twitter.com/chrispels)

In diesem video Chris Pels wird gezeigt, wie zum Erstellen einer Komponente, die auf mehreren Webseiten und Websites verwendet werden kann, die e-Mail-Nachrichten an eine Liste der Empfänger sendet. Zunächst wird der ASP.NET-Website konfiguriert werden, zum Senden von e-Mail mithilfe der &lt;MailSettings&gt; in der Datei "Web.config". Klicken Sie dann werden eine Klasse und mehrere Methoden zum Lesen einer Liste von Empfängern aus einer Datenquelle (DB, XML usw.), und Senden einer e-Mail-Nachricht an jeden der mithilfe der Klassen System.Net.Mail Empfänger erstellt. Als Teil dieses Prozesses Ausnahme ist enthalten Behandlung. Darüber hinaus wird eine Benutzeroberfläche erstellt, um die Elemente angeben, den der Benutzer kann z. B. die Absenderadresse für die Themenbereichsdatenbank, eine Anlage usw. hinzufügen.

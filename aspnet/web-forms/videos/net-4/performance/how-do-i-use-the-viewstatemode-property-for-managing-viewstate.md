====================
durch [Chris PEL-Spareinlagen](https://twitter.com/chrispels)

In diesem Video [Chris Pels](http://www.idevtech.com) zeigt, wie die Eigenschaft ViewStateMode beim Entwerfen einer Website in ASP.NET "ViewState" speichern. Zunächst wird die Funktionalität zum Verwalten von "ViewState" Speichern in ASP.NET vor ASP.NET 4 erläutert. Die Bedeutung der hierarchischen Vererbung der Eigenschaft ViewStateMode wird erläutert. Schließlich wird ein Beispielszenario für das Minimieren von "ViewState" Speichern auf ASP.NET-Seiten veranschaulicht.

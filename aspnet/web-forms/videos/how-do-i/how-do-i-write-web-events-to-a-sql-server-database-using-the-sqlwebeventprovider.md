<a name="how-do-i-write-web-events-to-a-sql-server-database-using-the-sqlwebeventprovider"></a>[Gewusst wie:] Schreiben Sie Webereignisse in einer SQL Server-Datenbank mithilfe der SqlWebEventProvider
====================
durch [Chris PEL-Spareinlagen](https://twitter.com/chrispels)

In diesem video Chris Pels zeigt, wie die ASP.NET-Systemüberwachung SqlWebEventProvider Protokollieren von Fehlern in einer Website mit einer SQL Server-Datenbank. Erfahren Sie zunächst die Rolle des Anbieters und Ereignisse in ASP.NET-Systemüberwachung aus. Im folgenden finden Sie zum Konfigurieren von SQL Server-Datenbank mit der erforderlichen Objekte für die Aufzeichnung Systemüberwachungsereignissen mit dem Aspnet\_Regiis Dienstprogramm, das gleiche verwendet, um die ASP.NET-Mitgliedschaft konfigurieren. Danach erfahren Sie in der Datei "Web.config" zum Aufzeichnen von Ereignissen in einer ASP.NET Web-Standort und die neu erstellte SQL Server-Datenbank für die Integritätsüberwachung konfigurieren. Im Rahmen dieser Konfiguration finden Sie unter der Stammdatei "Web.config" in .NET Framework 2.0 wie hat, Anbieter und Ereignisse definiert, die beim Konfigurieren der Überwachung des Integritätsstatus genutzt werden können. Diese Grundlagen werden eine Basis von der benutzerdefinierte Ereignisse erstellt werden können, die eigene spezifische Informationen zu einer SQL Server-Datenbank in eine ASP.NET-Website zu melden.

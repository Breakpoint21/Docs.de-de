<a name="how-do-i-use-the-conditional-updatemode-of-the-updatepanel"></a>[Gewusst wie:] Verwenden Sie die bedingte UpdateMode von UpdatePanel?
====================
durch [Joe Stagner](https://github.com/JoeStagner)

UpdatePanel von ASP.NET AJAX umfasst eine UpdateMode-Eigenschaft, die auf "Immer" oder 'Bedingte' festgelegt werden kann. Der Standardwert ist immer, die in diesem Fall die UpdatePanel immer seinen Inhalt während eines asynchrone Postbacks aktualisiert wird. In diesem Video lernen wir an, wie wir die UpdateMode auf bedingten, festlegen können in dem Fall UpdatePanel nur seinen Inhalt zu aktualisieren wird bei der serverseitigen Code der Update-Methode aufruft. Dies ermöglicht Ihnen die Verwendung von Bedingungslogik in Ihrem C#- oder Visual Basic-Code um zu bestimmen, ob die UpdatePanel seinen Inhalt während des aktuellen asynchronen Postbacks aktualisiert wird.

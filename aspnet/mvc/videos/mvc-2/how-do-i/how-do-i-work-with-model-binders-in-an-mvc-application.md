<a name="how-do-i-work-with-model-binders-in-an-mvc-application"></a>Gewusst wie: Arbeiten mit Modellbinder in MVC-Anwendung?
====================
durch [Chris PEL-Spareinlagen](https://twitter.com/chrispels)

In diesem video Chris Pels zeigt, wie Modellbinder zum Binden von Daten an komplexe Geschäftsobjekte in Modellen in einer MVC-Anwendung. Zuerst wird eine MVC-Anwendung sowie ein einfaches Modell für die Klasse "Kunde" zusammen mit einem Controller und Ansicht zum Bearbeiten von Kundendaten erstellt. Anschließend wird die automatische wurden die modellbindung von MVC anhand der Customer-Klasse veranschaulicht. Bind-Attribut ist als Nächstes erläutert und veranschaulicht, die als Mechanismus für die Details des Bindungsvorgangs ein-und Ausschließen von bestimmte Eigenschaften eines komplexen Objekts steuern. Schließlich wird die Verwendung von Namespace-Präfixe für die Segmentierung der Daten auf einer Seite erläutert.

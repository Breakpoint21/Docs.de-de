====================
durch [Chris PEL-Spareinlagen](https://twitter.com/chrispels)

In diesem video Chris Pels zeigt, wie die Attribute HttpVerbs zum Steuern des Zugriffs für MVC-Aktionen. Zuerst wird eine beispielanwendung, mit einem Standard-Controller und Ansicht zum Bearbeiten der Informationen erstellt. Als Nächstes wird eine zweite Index-Aktion an den Controller hinzugefügt, die ein HttpPost-Attribut hat die einschränkt, er wird aufgerufen, wenn eine HTTP POST verwendet wird. Als eine nachverfolgung erforderlich ist wird das Attribut AcceptVerbs() für Visual Studio 2008 als eine alternative Syntax implementiert. Die HttpVerbs für verhindert, dass das Sicherheitsrisiko zugeordnet, die zum Ausführen einer Delete über eine Verknüpfung mithilfe einer HTTP GET verwendet wird anschließend erläutert.

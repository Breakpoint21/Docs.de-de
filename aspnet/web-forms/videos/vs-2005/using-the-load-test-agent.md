<a name="using-the-load-test-agent"></a>Verwenden der Auslastungstest-Agent
====================
von Eric Lee

In diesem Video untersuchen wir wie der Load Test-Agent mit dem Testen, die wir mit Visual Studio Team System integriert. Eine laden-Test-Agent ist Teil eines "Rigs", das zum Trennen von Computern zum Laden, Spannung und Komponententests aus einem anderen Betriebssystem System- oder Computer-Konfiguration können bereitgestellt werden kann. Jede Rigs kann für mehrere Agents und kann für virtuelle oder remote-Computer, bei dem eine umfassende Tests mit Visual Studio Team System bereitgestellt werden.

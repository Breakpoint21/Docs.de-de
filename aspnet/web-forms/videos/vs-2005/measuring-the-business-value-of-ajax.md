<a name="measuring-the-business-value-of-ajax"></a>Messen Sie den Geschäftswert AJAX
====================
von Eric Lee

ASP.NET AJAX ist gut bekannt für die Fähigkeit zur Verbesserung der benutzerfreundlichkeit von Webanwendungen. Mit Visual Studio Team System können Sie auch den Geschäftswert Hinzufügen von ASP.NET AJAX zu Ihren Webanwendungen messen. Insbesondere können Sie das Laden und die Ausführung von Webtests Funktionen von Visual Studio Team System verwenden, um zu untersuchen, wie der Unterschied in der HTTP-Übertragung Datengröße wirkt sich die Leistung und Skalierbarkeit Merkmale der Webanwendung auf.

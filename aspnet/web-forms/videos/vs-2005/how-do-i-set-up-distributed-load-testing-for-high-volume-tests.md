<a name="how-do-i-set-up-distributed-load-testing-for-high-volume-tests"></a>[Gewusst wie:] Richten Sie verteilte Auslastungstests für hohes Volumen Tests?
====================
durch [Chris Menegay](https://twitter.com/CMenegay)

Visual Studio Team System können Sie verteilte Auslastungstests durch Erstellen eines Tests "Lichtanlage" einrichten, die einen Controller auf einem Remotecomputer und mehrere Agents auf den gleichen oder zusätzlichen Remotecomputern umfasst. Dieses Video führt Sie durch die Installation der Controller und den Agents, und klicken Sie dann Konfigurierung von Testfälle über der Remotecontroller ausgeführt werden. Wie Sie sehen werden, erleichtert Visual Studio Team System zum Einrichten der verteilten Auslastungstests für hohes Volumen automatisierte Tests sehr einfach.

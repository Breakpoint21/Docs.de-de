<span data-ttu-id="be392-153">Wenn Sie Benutzereingaben akzeptieren, und die Eingabe auf Ihrer Website anzuzeigen, öffnen Sie möglicherweise Ihre Website für Cross-Site-Skripting (XSS)-Angriffe.</span><span class="sxs-lookup"><span data-stu-id="be392-153">Whenever you accept input from a user, and redisplay that input on your website, you potentially open your website to Cross-Site Scripting (XSS) Attacks.</span></span> <span data-ttu-id="be392-154">Theoretisch kann ein böswilliger Hacker JavaScript-Code senden, der ausgeführt wird, wenn die Eingabe erneut angezeigt wird.</span><span class="sxs-lookup"><span data-stu-id="be392-154">In theory, a malicious hacker could submit JavaScript code that gets executed when the input is redisplayed.</span></span> <span data-ttu-id="be392-155">Der JavaScript-Code kann verwendet werden, Kennwörter oder andere sensible Daten zu stehlen.</span><span class="sxs-lookup"><span data-stu-id="be392-155">The JavaScript could be used to steal user passwords or other sensitive information.</span></span>

====================
durch [Chris Menegay](https://twitter.com/CMenegay)

Es gibt eine Reihe von continuous Integrationstools, die zusätzlich zu Team Foundation Server arbeiten, und in diesem Video untersuchen wir die kostenlose [Norton TeamCI](http://teamsystemrocks.com/files/12/tools/entry1018.aspx) Tool verfügbar [Team System Rocks](http://teamsystemrocks.com/). TeamCI überprüft in regelmäßigen Abständen, ob aktualisiert, Dateien Team Foundation Server überprüft wurden und, wenn dies der Fall ist, erstellen Sie die Projektmappe und automatisierte Tests ausgeführt werden.

<a name="how-do-i-event-handlers-in-aspnet-web-pages-and-controls"></a>[Gewusst wie:] -Ereignishandler in ASP.NET Web Pages und Steuerelemente
====================
durch [Chris PEL-Spareinlagen](https://twitter.com/chrispels)

In diesem video Chris Pels erfahren, wie Ereignishandler für ASP.NET Web Pages und Webserversteuerelemente erstellen. Zunächst Informationen Sie zum Erstellen von Seite-Ereignissen für die Seite "standard"-Ereignisse, die von .NET Framework bereitgestellt werden. Informationen Sie zum Binden Sie die Ereignisse automatisch auf eine Seite mit dem AutoEventWireup-Attribut als auch programmgesteuert binden Sie die Ereignisse. Als Nächstes erfahren Sie, wie erstellen und binden Ereignishandler für die standard-Web-Server-Steuerelementereignisse. Erfahren Sie außerdem, wie mehrere Ereignisse für andere Steuerelemente an den gleichen Ereignishandler eine Verbindung herstellen. Schließlich finden Sie die Verwendung der Eigenschaft AutoPostbackTrue für Webserversteuerelemente aus.

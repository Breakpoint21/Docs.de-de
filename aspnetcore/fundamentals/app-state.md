<span data-ttu-id="ce021-109">Sitzungszustand ist ein Feature in ASP.NET Core, das Sie verwenden können, um Benutzerdaten zu speichern, während der Benutzer Ihre Web-App durchsucht.</span><span class="sxs-lookup"><span data-stu-id="ce021-109">Session state is a feature in ASP.NET Core that you can use to save and store user data while the user browses your web app.</span></span> <span data-ttu-id="ce021-110">Der Sitzungszustand besteht aus einem Wörterbuch oder einer Hashtabelle auf dem Server und speichert Daten für mehrere Anforderungen, die von einem Browser gesendet werden.</span><span class="sxs-lookup"><span data-stu-id="ce021-110">Consisting of a dictionary or hash table on the server, session state persists data across requests from a browser.</span></span> <span data-ttu-id="ce021-111">Die Sitzungsdaten werden in einem Cache gesichert.</span><span class="sxs-lookup"><span data-stu-id="ce021-111">The session data is backed by a cache.</span></span>

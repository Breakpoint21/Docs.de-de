<a name="how-do-i-implement-the-persistent-communications-pattern-with-the-updatepanel"></a>[Gewusst wie:] Implementiert das Muster beständigen Kommunikation mit dem UpdatePanel?
====================
durch [Joe Stagner](https://github.com/JoeStagner)

In einer herkömmlichen Website im Browser und dem Server eine laufende Kommunikation nicht beibehalten, aber kommunizieren nur als Reaktion auf der Benutzer eine Aktion ausführt. In einer modernen-Website, wo die Seite für einen Anwendungscontainer wird, kann es vorteilhaft sein für Browser und dem Server eine laufende Kommunikation verwaltet werden, sodass Seitenupdates auftreten können, ohne dass der Benutzer eine Aktion aus. Dies wird als permanente Kommunikation Muster für AJAX bezeichnet. ASP.NET AJAX bietet zwei verschiedene Arten für Webentwickler, die permanenten Kommunikation Muster zu implementieren. Dieses Video zeigt die einfache Möglichkeit, um ASP.NET AJAX UpdatePanel als Grundlage für die Implementierung zu verwenden. In einem späteren Zeitpunkt video erfahren wir, wie dasselbe Muster, das ohne die Verwendung von ASP.NET AJAX UpdatePanel implementiert.

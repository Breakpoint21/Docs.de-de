<a name="how-do-i-use-the-aspnet-ajax-updatepanelanimation-extender"></a>[Gewusst wie:] Verwenden des ASP.NET AJAX UpdatePanelAnimation Extender?
====================
durch [Joe Stagner](https://github.com/JoeStagner)

Durch Hinzufügen von Animationen zu einem Teilrendering von Seiten Update wir den Benutzer zu informieren können, wenn ein Update stattfindet, und wurde abgeschlossen. Der Extender UpdatePanelAnimation hinzugefügt UpdatePanel von ASP.NET AJAX-Steuerelement, und Nutzen von umfangreichen Animation Framework Core ASP.NET AJAX-Erweiterungen Animationseffekte. Dieses Video zeigt, wie Sie die Reihenfolge der Effekte für das UpdatePanelAnimation definieren können, und wie Sie mithilfe von ScriptAction Definitionen die Animation optimieren können.

## <a name="configuring-web-server-permissions"></a><span data-ttu-id="b5ebc-119">Konfigurieren von Webserverberechtigungen</span><span class="sxs-lookup"><span data-stu-id="b5ebc-119">Configuring Web Server Permissions</span></span>

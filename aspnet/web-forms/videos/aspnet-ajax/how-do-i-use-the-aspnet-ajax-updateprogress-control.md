<a name="how-do-i-use-the-aspnet-ajax-updateprogress-control"></a>[Gewusst wie:] Verwenden Sie das ASP.NET AJAX-Steuerelement UpdateProgress?
====================
durch [Joe Stagner](https://github.com/JoeStagner)

In einer AJAX-Anwendung ist häufig die merkliche Pause zwischen dem Zeitpunkt, den der Browser Netzwerk anfordert und die Zeit, die der Browser eine Antwort erhält. Damit den Benutzer wissen, dass die netzwerkkonversation, die ASP.NET AJAX-UpdateProgress stattfindet bietet Steuerung unmittelbares Feedback, indem Sie anzeigen, eine animierte Grafik, Text "laden" oder anderen HTML-Code, der nicht mehr angezeigt wird, sobald die Netzwerkantwort empfangen wird.

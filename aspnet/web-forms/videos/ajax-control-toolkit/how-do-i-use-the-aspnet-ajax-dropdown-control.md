<a name="how-do-i-use-the-aspnet-ajax-dropdown-control"></a>[Gewusst wie:] Verwenden Sie das DropDown-Steuerelement in ASP.NET AJAX?
====================
durch [Joe Stagner](https://github.com/JoeStagner)

Mit dem DropDown-Steuerelement-Extender aus dem ASP.NET AJAX-Steuerelement-Toolkit können Sie den Benutzer mit einer Dropdown-Liste der Optionen darstellen, die ein Update Teilrendering von Seiten an anderer Stelle auf der Webseite auslösen. Die vom Benutzer ausgewählten Option wird auf dem Server, sodass serverseitigen Code die Auswahl vor der Durchführung des Updates Teilrendering von Seiten verarbeitet übergeben.
